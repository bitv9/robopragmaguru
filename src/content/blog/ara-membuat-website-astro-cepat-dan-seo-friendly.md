## Cara Membuat Website Astro yang Cepat dan SEO Friendly

Astro menjadi salah satu framework modern yang banyak digunakan untuk membangun website cepat dan ringan.

Framework ini cocok digunakan untuk:

- Landing page
- Blog
- Company profile
- Documentation
- Website SEO friendly

Karena Astro menggunakan pendekatan static-first, website dapat memiliki loading lebih cepat dibanding banyak framework lain.

---

## Mengapa Menggunakan Astro?

Astro dirancang untuk meminimalkan JavaScript yang dikirim ke browser.

Keunggulan Astro:

- Loading cepat
- Struktur sederhana
- SEO friendly
- Build ringan
- Performa mobile lebih baik

Pendekatan ini membantu meningkatkan user experience dan performa website secara keseluruhan.
